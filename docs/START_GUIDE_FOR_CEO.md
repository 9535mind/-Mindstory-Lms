- 학생들이 과정을 신청하고 **실제로 돈을 결제**할 수 있게 해줍니다
- 환불도 자동으로 처리됩니다

### 무엇을 하나요?

#### 📍 Step 1: 회원가입 (10분)
1. 컴퓨터로 이 주소 접속: **https://www.tosspayments.com**
2. 오른쪽 위 **"무료로 시작하기"** 클릭
3. 이메일로 회원가입: **sanj2100@naver.com**
4. 비밀번호 설정 (잊어버리지 않게 메모!)

#### 📍 Step 2: 사업자 인증 (1-2일 소요)
가입 후 화면에서 **"사업자 인증하기"** 버튼 클릭

**필요한 서류** (스캔 또는 사진으로 제출):
```
✅ 사업자등록증 사본
   → 국세청 홈택스에서 출력 가능
   → 또는 세무서에서 발급

✅ 통장 사본
   → 농협 351-1202-0831-23
   → 인터넷뱅킹에서 출력 가능

✅ 대표자 신분증 사본
   → 주민등록증 또는 운전면허증
```

**제출 방법**:
- 토스페이먼츠 사이트에서 파일 업로드
- 또는 이메일로 전송

**승인 시간**: 1-2 영업일 (평일 기준)

#### 📍 Step 3: 정산 계좌 등록 (승인 후)
사업자 인증 승인되면:
1. "정산 관리" 메뉴 클릭
2. 계좌 정보 입력
   - 은행: 농협
   - 계좌번호: 351-1202-0831-23
   - 예금주: (주)마인드스토리
3. 계좌 인증 (소액 입금 확인)

#### 📍 Step 4: API 키 받기 (승인 후 즉시)
1. "개발자 센터" 메뉴 클릭
2. "API 키 발급" 클릭
3. **두 개의 키를 복사해서 저에게 전달**:
   ```
   클라이언트 키: toss_ck_실제키값...
   시크릿 키: toss_sk_실제키값...
   ```

### 왜 필요한가요?

#### 📍 Step 1: 무료 상담 신청 (10분)
1. 컴퓨터로 이 주소 접속: **https://www.kollus.com**
2. "무료 체험 신청" 또는 "문의하기" 클릭
3. 정보 입력:
   ```
   회사명: (주)마인드스토리
   담당자: 박종석
   전화: 062-959-9535
   이메일: sanj2100@naver.com
   서비스: 교육 콘텐츠 (원격평생교육원)
   ```

#### 📍 Step 2: 영업팀과 통화 (1-2일 내)
Kollus 영업팀에서 전화 옵니다:
- 어떤 서비스인지 설명해줍니다
- 가격 안내 (월 30만원 ~)
- 무료 체험 제공 (보통 14일)

**미리 준비할 것**:
```
✅ 예상 학생 수: 몇 명 정도?
✅ 영상 개수: 몇 개 강의?
✅ 평균 영상 길이: 한 강의당 몇 분?
```

#### 📍 Step 3: 계약 및 가입 (통화 후)
- 사업자등록증 제출
- 계약서 작성 (온라인 가능)
- 계정 생성 (아이디/비밀번호 받음)

#### 📍 Step 4: 영상 업로드
1. Kollus 관리자 페이지 로그인
2. "미디어 업로드" 클릭
3. 촬영한 강의 영상 파일 업로드
4. 인코딩 대기 (10분~1시간)
5. 영상 URL 받기

**영상 준비 팁**:
```
✅ 형식: MP4 (가장 일반적)
✅ 화질: 1080p 권장 (핸드폰 촬영도 OK)
✅ 길이: 10-30분씩 나누기
✅ 파일명: "1강-마인드타임이란.mp4" 형식
```
