### 백업 복구 방법
```bash
# 1. 백업 파일 다운로드
wget https://www.genspark.ai/api/files/s/Od5gG04K -O mindstory-lms-v1.3.3.tar.gz

# 2. 압축 해제
tar -xzf mindstory-lms-v1.3.3.tar.gz

# 3. 디렉토리 이동
cd webapp

# 4. 의존성 설치
npm install

# 5. 데이터베이스 초기화
rm -rf .wrangler/state/v3/d1
npx wrangler d1 migrations apply mindstory-production --local
npx wrangler d1 execute mindstory-production --local --file=./seed.sql

# 6. 빌드
npm run build

# 7. 서비스 시작
pm2 start ecosystem.config.cjs

# 8. 확인
curl http://localhost:3000/api/health
```
