## ⏰ 시간별 테스트 일정

### Phase 1: 인증 시스템 (30분)
- 회원가입 (이메일/Google/Kakao)
- 로그인/로그아웃
- 프로필 수정
- 비밀번호 변경
- 회원 탈퇴

### Phase 2: 강좌 시스템 (30분)
- 강좌 목록 조회
- 추천 강좌 표시
- 강좌 상세 조회
- 차시 목록 조회

### Phase 3: 수강 시스템 (1시간)
- 무료 강좌 즉시 수강
- 유료 강좌 결제 필요 안내
- 내 강의실 조회
- 진도율 표시

### Phase 4: 영상 시스템 (1시간) ⚠️
- **현재 미구현**: video_url null
- **권장 조치**: YouTube Private 영상 사용

### Phase 5: 결제 시스템 (1시간)
- 결제 생성 (Toss 테스트 환경)
- 결제 승인
- 환불 처리

### Phase 6: 수료증 시스템 (30분)
- 진도율 80% 이상 확인
- 수료증 자동 발급
- 수료증 조회

### Phase 7: 관리자 시스템 (1시간)
- 대시보드 통계
- 강좌 생성/수정/삭제
- 회원 관리
- 결제 관리

### Phase 8: 통합 시나리오 (1시간)
- End-to-End 테스트
- 신규 사용자 → 수료증 발급까지

### Phase 9: 문서화 (1시간)
- 테스트 결과 정리
- 이슈 리포트
- 최종 체크리스트
