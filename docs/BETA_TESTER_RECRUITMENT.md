## 📋 베타 테스트 가이드

### 테스트 시나리오

#### 시나리오 1: 신규 회원가입 → 무료 강좌 수강
```
소요 시간: 1시간
난이도: ★☆☆☆☆

1. 홈페이지 접속
2. [회원가입] 클릭
3. Google 또는 카카오 로그인
4. "무료 체험 과정" 찾기
5. [수강 신청] 클릭
6. "내 강의실" 확인
7. 첫 번째 차시 시청
```

#### 시나리오 2: 유료 강좌 결제 → 학습
```
소요 시간: 2-3시간
난이도: ★★☆☆☆

1. "자기주도학습 지도사 과정" 선택
2. [수강 신청] 클릭
3. 테스트 카드로 결제
   - 카드번호: 4000-0000-0000-0001
   - 유효기간: 2030년 12월
   - CVC: 123
4. 결제 완료 확인
5. "내 강의실"에서 학습 시작
6. 최소 3개 차시 시청
7. 진도율 확인
```

#### 시나리오 3: 관리자 화면 체험 (선택)
```
소요 시간: 30분
난이도: ★★★☆☆

⚠️ 관리자 계정 별도 제공 필요

1. 관리자 로그인
2. 대시보드 확인
3. [관리자 모드] ↔ [수강생 모드] 전환
4. 통계 확인
5. 회원 관리 페이지 확인
```
