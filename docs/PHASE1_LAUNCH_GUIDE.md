## 📋 **1단계: Cloudflare R2 버킷 생성 (10분)**

### **① Cloudflare 대시보드 접속**
1. https://dash.cloudflare.com 로그인
2. 좌측 메뉴 **R2** 클릭

### **② 첫 번째 버킷 생성 (영상용)**
```
Bucket name: mindstory-videos
Location: Automatic (자동 선택)
```
- **Create bucket** 클릭

### **③ 두 번째 버킷 생성 (문서용)**
```
Bucket name: mindstory-storage
Location: Automatic
```
- **Create bucket** 클릭

### **④ API 토큰 생성**
1. R2 페이지 → **Manage R2 API Tokens**
2. **Create API token** 클릭
3. 설정:
   ```
   Token name: mindstory-lms-token
   Permissions: Object Read & Write
   Buckets: mindstory-videos, mindstory-storage
   TTL: Forever
   ```
4. **Create API token** 클릭
5. **⚠️ 중요: 다음 정보 저장 (다시 안 보임)**
   ```
   Access Key ID: xxxxxxxxxxxxx
   Secret Access Key: yyyyyyyyyyyy
   Account ID: zzzzzzzzzzzz
   ```

## 📋 **3단계: 첫 강의 영상 준비 (30분)**

### **① 영상 촬영/편집 완료**
- 강의 영상 MP4 파일 준비

### **② 영상 인코딩 (필수)**
**권장 설정 (HandBrake 기준):**
```
포맷: MP4
비디오 코덱: H.264
해상도: 1920x1080 (Full HD)
프레임레이트: 30fps
비트레이트: 3-5 Mbps
오디오: AAC, 128 kbps
```

**예상 파일 크기:**
- 10분 강의: 약 50MB
- 30분 강의: 약 150MB
- 1시간 강의: 약 300MB

### **③ 영상 업로드 (Cloudflare R2)**

**방법 A: wrangler CLI (추천)**
```bash
cd /home/user/webapp

# 영상 업로드
npx wrangler r2 object put mindstory-videos/videos/lesson-001.mp4 \
  --file=/path/to/your/video.mp4

# 업로드 확인
npx wrangler r2 object list mindstory-videos
```

**방법 B: 관리자 페이지 (개발 예정)**
- 추후 드래그앤드롭 업로드 UI 추가 가능

## 📋 **4단계: 관리자 대시보드에서 강좌 등록 (10분)**

```

     https://your-app.pages.dev/login (운영)
```

### **② 강좌 생성**
1. 관리자 대시보드 → **강좌 관리**
2. **새 강좌 추가** 클릭
3. 입력:

2. **차시 관리** 클릭
3. **새 차시 추가**
4. 입력:
   ```

   제목: 1강. 마인드 타임이란?
   영상 공급자: r2
   영상 키: videos/lesson-001.mp4
   영상 URL: (자동 생성됨)
   재생 시간: 30분
   무료 체험: 활성화 (첫 강의만)
   ```

## 🚨 **문제 발견 시 즉시 대응 프로세스**

### **① 영상 재생 안 됨**
**증상:** 영상이 로딩만 되고 재생 안 됨

**원인 가능성:**
1. R2 바인딩 설정 오류
2. 영상 파일 인코딩 문제
3. 네트워크 이슈

**해결 방법:**
```bash
# 1. R2 바인딩 확인
wrangler.jsonc 확인

# 2. 영상 파일 재인코딩
HandBrake로 MP4 H.264 재생성

# 3. 브라우저 콘솔 확인
F12 → Console → 에러 메시지 확인
```

**긴급 연락:** 제게 바로 문의!

---

### **② 진도율이 안 올라감**
**증상:** 영상은 재생되는데 진도율 0%

**원인:**
1. JavaScript 오류
2. 세션 만료
3. API 통신 오류

**해결:**
```bash
# 1. 브라우저 콘솔 확인
F12 → Console → 에러 확인

# 2. 네트워크 탭 확인
F12 → Network → /api/progress/update 요청 확인

# 3. 로그아웃 후 재로그인
```

---

### **③ 영상 다운로드 시도 발견**
**증상:** 수강생이 영상 다운로드 시도

**대응:**
1. **즉시 경고 이메일 발송**
2. **계정 일시 정지**
3. **약관 위반 경고**
4. **재발 시 영구 정지**

**예방:**
- 회원가입 시 약관 명시 강화
- "영상 무단 다운로드/공유 금지" 팝업

---

### **④ 수료증 발급 오류**
**증상:** 수료했는데 수료증 버튼 안 보임

**원인:**
1. 진도율 80% 미만
2. DB 동기화 오류

**해결:**
```sql
-- 관리자가 직접 수료 처리
UPDATE enrollments 
SET is_completed = 1, completed_at = datetime('now')
WHERE user_id = ? AND course_id = ?;
```
