# 🏆 민간자격 시스템 가이드

## 마인드스토리 LMS - 민간자격 연계 기능

수료 후 **민간자격 취득**까지 원스톱으로 지원하는 시스템입니다.

---

## 📋 시스템 개요

### 주요 기능
1. **자격 종류 관리** - 관리자가 민간자격 등록
2. **강좌-자격 연계** - 특정 강좌 수료 시 자격 신청 가능
3. **자동 안내 팝업** - 수료 완료 시 자격 안내 표시
4. **신청 프로세스** - 학생이 온라인으로 자격 신청
5. **심사 시스템** - 관리자가 신청 검토 및 승인
6. **자격증 발급** - 승인 후 자격증 번호 발급

---

## 🎯 사용자별 기능

### 학생 (수강생)
1. ✅ 강좌 수료 완료
2. 🎉 자동으로 민간자격 안내 팝업 표시
3. 📝 자격 요건 확인
4. 🚀 온라인 신청

3. 📋 신청 내역 조회
4. 🔍 신청서 검토
5. ✅ 승인 또는 반려
6. 📜 자격증 발급

## 🚀 빠른 시작

### 1단계: 민간자격 등록 (관리자)

**한국직업능력연구원 등록 먼저!**
```
1. https://www.pqi.or.kr 접속
2. 민간자격 등록 신청
3. 등록번호 발급 받기 (예: 2024-000123)
```

**LMS에 자격 등록**
```
관리자 로그인 → 설정 → 민간자격 관리 → 새 자격 추가

필수 정보:
- 자격명: "감정코칭 전문가 2급"
- 자격 코드: "EC-L2"
- 분류: coaching / counseling / psychology
- 등록번호: 2024-000123 (한국직업능력연구원)
- 발급 비용: 50,000원
- 유효 기간: 0 (평생) 또는 24개월 등
```

### 2단계: 강좌와 연결

```
민간자격 상세 페이지 → 연결 강좌 설정

예시:
"감정코칭 전문가 2급" ← "감정코칭 전문가 과정" (필수)
```

### 3단계: 학생이 수료 완료하면?

**자동으로 다음이 실행됩니다:**
1. 수료 처리
2. 자격 안내 팝업 자동 표시
3. 학생이 원클릭으로 신청 가능

## 📊 신청 프로세스

### 신청 상태 흐름

```
📝 pending (신청 대기)
  ↓
🔍 reviewing (검토 중) - 선택사항
  ↓
✅ approved (승인) 또는 ❌ rejected (반려)
  ↓
🎓 issued (발급 완료)
```

### 상태별 설명

**pending** - 신청 완료, 관리자 검토 대기
- 학생이 신청서 제출
- 관리자 대시보드에 알림

**reviewing** - 관리자가 검토 중
- 선택사항 (바로 approved/rejected 가능)
- 복잡한 경우 사용

**approved** - 승인 완료
- 관리자가 요건 확인 후 승인
- 이제 자격증 발급 가능

**rejected** - 반려
- 요건 미충족 등
- 반려 사유 기록

**issued** - 자격증 발급 완료
- 자격증 번호 생성
- PDF 자격증 다운로드 가능 (TODO)
- 만료일 설정 (평생/기간제)

## 🔗 API 엔드포인트

### 공개 API

**GET /api/certifications/types**
- 민간자격 목록 조회

**GET /api/certifications/types/:id**
- 특정 자격 상세 정보 (연계 강좌 포함)

### 학생 API (로그인 필요)

**GET /api/certifications/eligible/:typeId**
- 자격 신청 가능 여부 확인

**POST /api/certifications/apply**
- 자격 신청

**GET /api/certifications/my-applications**
- 내 신청 내역

**GET /api/certifications/application/:id**
- 특정 신청 상세 정보

**POST /api/certifications/application/:id/cancel**
- 신청 취소

### 관리자 API

**GET /api/admin/certifications/applications**
- 전체 신청 목록

**POST /api/admin/certifications/applications/:id/review**
- 신청 심사 (승인/반려)

**POST /api/admin/certifications/applications/:id/issue**
- 자격증 발급

**GET /api/admin/certifications/statistics**
- 통계 조회

```javascript

// 3. 자격증 발급

{
  "success": true,
  "data": {
    "certificate_number": "CERT-EC-L2-2025-0001",
    "issue_date": "2025-12-28T10:30:00Z"
  }
}
```
