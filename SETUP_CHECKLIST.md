# 🚀 프로덕션 배포 체크리스트

## ✅ 사전 준비 (5분)

### 1단계: Cloudflare 가입 (2분)
- [ ] https://dash.cloudflare.com/sign-up 접속
- [ ] 이메일 + 비밀번호로 가입
- [ ] 이메일 인증 완료
- [ ] Free 플랜 선택

### 2단계: API 토큰 생성 (2분)
- [ ] Cloudflare 대시보드 로그인
- [ ] 프로필 → "My Profile" 클릭
- [ ] "API Tokens" 메뉴 선택
- [ ] "Create Token" 클릭
- [ ] "Edit Cloudflare Workers" 템플릿 사용
- [ ] 토큰 생성 후 복사 (**한 번만 표시됨!**)

### 3단계: API 키 설정 (1분)
- [ ] Deploy 탭에서 API 키 입력
- [ ] 저장 버튼 클릭

---

## 🎯 배포 실행 (자동, 5분)

API 키 설정 완료 후, 저에게 다음과 같이 말씀해주세요:

```
배포해줘!
```

또는

```
프로덕션 배포 시작!
```

그러면 자동으로:
1. ✅ 프로젝트 빌드
2. ✅ R2 스토리지 생성 (영상 저장용)
3. ✅ D1 데이터베이스 생성
4. ✅ Cloudflare Pages 배포
5. ✅ 데이터베이스 마이그레이션
6. ✅ 초기 데이터 입력

---

## 📊 배포 완료 후

배포가 완료되면 받게 되는 것:

1. **프로덕션 URL**
   ```
   https://xxx.mindstory-lms.pages.dev
   ```

2. **관리자 로그인**
   - 이메일: `admin@mindstory.co.kr`
   - 비밀번호: `admin123`

3. **영상 업로드 방법**
   - 관리자 로그인
   - 강좌 관리 → 차시 관리
   - 영상 파일 선택 (최대 500MB)
   - 저장 (자동으로 R2에 업로드)

---

## 🆘 문제 해결

### API 토큰을 잃어버렸어요
→ Cloudflare 대시보드에서 새로 생성하세요 (2분)

### 배포가 실패했어요
→ 에러 메시지를 저에게 알려주세요 (즉시 해결)

### 영상 업로드가 안 돼요
→ 관리자 계정으로 로그인했는지 확인하세요
