# 기존 토큰 D1 권한 확인 가이드

## 현재 상황
"Cloudflare Workers 편집" 토큰이 이미 있습니다.
이 토큰에 D1 권한이 있는지 확인해야 합니다.

---

## 확인 방법

### 1단계: 토큰 상세 보기
1. "Cloudflare Workers 편집" 행의 오른쪽 **점 3개 (⋮)** 클릭
2. **"편집" (Edit)** 또는 **"보기" (View)** 선택

### 2단계: 권한 확인
권한 목록에서 다음을 찾아보세요:
- ✅ **D1** → **Edit** (있으면 OK!)
- ❌ D1이 없으면 → 새 토큰 필요

---

## 결과에 따라:

### ✅ D1 권한이 있는 경우:
→ "D1 권한 있어!" 라고 알려주세요
→ 기존 토큰 사용 (새로 만들 필요 없음)

### ❌ D1 권한이 없는 경우:
→ "D1 권한 없어!" 라고 알려주세요
→ 새 토큰 생성 필요
