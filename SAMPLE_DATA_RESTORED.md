## 🎥 영상 정보

### 영상 형식
- **타입**: YouTube 영상
- **video_type**: 'youtube'
- **video_url**: YouTube 비디오 ID 포함

### 무료 체험
- 각 강좌의 **첫 번째 차시는 무료**로 제공
- 수강 신청 전에 미리보기 가능

## 📊 데이터베이스 구조

### 새로 추가된 테이블

#### lessons 테이블
```sql
- id: 차시 ID
- course_id: 강좌 ID
- lesson_number: 차시 번호
- title: 차시 제목
- description: 차시 설명
- video_url: YouTube 비디오 ID
- video_type: 'youtube'
- duration_minutes: 재생 시간(분)
- is_free: 무료 여부 (0 또는 1)
- created_at, updated_at
```

#### lesson_progress 테이블
```sql
- id: 진도 기록 ID
- enrollment_id: 수강신청 ID
- lesson_id: 차시 ID
- status: 학습 상태
- watch_percentage: 시청률
- last_watched_at: 마지막 시청 시각
- is_completed: 완료 여부
- completed_at: 완료 시각
```
