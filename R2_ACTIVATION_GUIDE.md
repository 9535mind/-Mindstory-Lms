# R2 Storage 활성화 가이드

## 🎯 목표
Cloudflare R2 Storage를 활성화하여 영상 파일 업로드 기능 사용

---

## 📍 1단계: R2 페이지로 이동

### 방법 1: 직접 링크 (가장 빠름)
```
https://dash.cloudflare.com/2e8c2335c9dc802347fb23b9d608d4f4/r2
```

### 방법 2: 대시보드에서 이동
1. Cloudflare Dashboard 열기
2. 좌측 사이드바에서 **R2** 클릭
3. 또는 **Workers & Pages** → **R2** 클릭

---

## 📍 2단계: R2 활성화

### 화면에 보일 내용:
- "Enable R2" 버튼
- 또는 "Get Started" 버튼
- 무료 플랜 안내

### 할 일:
1. **"Enable R2"** 또는 **"Get Started"** 버튼 클릭
2. 무료 플랜 확인 (월 10GB 무료)
3. **"Continue"** 또는 **"Activate"** 클릭

---

## ✅ 완료 확인

활성화되면 다음이 보입니다:
- "Create bucket" 버튼
- 또는 "Your R2 buckets" 목록 화면

---

## 🎉 완료 후 알려주세요!

다음 중 하나로 알려주시면 됩니다:
- "R2 활성화했어!"
- "Enable 버튼 눌렀어!"
- "Create bucket 버튼 보여!"

그러면 바로 2단계로 넘어가겠습니다!

---

## ❓ 문제가 생기면?

- 결제 정보 요구: 카드 등록 필요 (무료 사용 가능)
- Enable 버튼 없음: 이미 활성화된 상태
- 에러 발생: 스크린샷 보내주세요
