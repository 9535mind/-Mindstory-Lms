# 📸 Cloudflare API 토큰 생성 - 시각적 가이드

## 🎬 화면별 안내

### 1️⃣ 시작 화면

로그인하면 보이는 Cloudflare 대시보드:

```
┌─────────────────────────────────────────────────────┐
│ Cloudflare Dashboard                    [👤 프로필]  │
├─────────────────────────────────────────────────────┤
│                                                     │
│  여기서 우측 상단 프로필 아이콘 클릭!                  │
│                                                     │
└─────────────────────────────────────────────────────┘
```

---

### 2️⃣ 프로필 메뉴

프로필 클릭하면 나타나는 드롭다운:

```
┌──────────────────────┐
│ My Profile      ← 클릭│
│ Account Settings     │
│ Billing              │
│ Sign Out             │
└──────────────────────┘
```

---

### 3️⃣ API Tokens 메뉴

좌측 사이드바에서:

```
┌─────────────────────┐
│ Profile              │
│ Communication        │
│ Authentication       │
│ API Tokens      ← 클릭│
│ Sessions             │
└─────────────────────┘
```

---

### 4️⃣ Create Token 버튼

API Tokens 페이지:

```
┌─────────────────────────────────────────────────┐
│ API Tokens                                      │
│                                                 │
│ User API Tokens                                 │
│                                                 │
│ [  + Create Token  ]  ← 이 파란색 버튼 클릭!      │
│                                                 │
└─────────────────────────────────────────────────┘
```

---

### 5️⃣ 템플릿 선택 화면

여러 템플릿 중에서:

```
┌───────────────────────────────────────────────┐
│ Create API Token                              │
├───────────────────────────────────────────────┤
│                                               │
│ Edit Cloudflare Workers                       │
│ Allows you to deploy Workers and Pages       │
│                                               │
│                         [Use template] ← 클릭  │
│                                               │
├───────────────────────────────────────────────┤
│                                               │
│ Read all resources                            │
│ ...                                           │
│                                               │
└───────────────────────────────────────────────┘
```

⚠️ **정확히 "Edit Cloudflare Workers"를 찾으세요!**

---

### 6️⃣ 권한 확인 화면

```

│    └─ Cloudflare Pages         [Edit ▼]      │

│ ✅ Account                                     │

│                                               │
│ ✅ Account                                     │

│                                               │
│                                               │
│              [Continue to summary]  ← 클릭     │
│                                               │
└───────────────────────────────────────────────┘
```

---
