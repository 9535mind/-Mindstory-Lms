# ✅ Cloudflare 가입 체크리스트

## 🎯 준비물
- [ ] 이메일 주소 (Gmail, Naver, Daum, 회사 이메일 등)
- [ ] 비밀번호 (8자 이상, 영문+숫자)

---

## 📝 가입 단계

### 1단계: 가입 페이지 접속
- [ ] 브라우저 열기
- [ ] https://dash.cloudflare.com/sign-up 접속

### 2단계: 정보 입력
- [ ] 이메일 주소 입력
- [ ] 비밀번호 입력 (8자 이상)
- [ ] 약관 동의 체크
- [ ] Sign Up 버튼 클릭

### 3단계: 이메일 인증
- [ ] 이메일 받은편지함 열기
- [ ] Cloudflare 이메일 찾기
- [ ] "Verify Email" 버튼 클릭
- [ ] 스팸함도 확인 (이메일이 안 오면)

### 4단계: 플랜 선택
- [ ] Free 플랜 선택 ($0/월)
- [ ] Select 버튼 클릭

### 5단계: 완료 확인
- [ ] Cloudflare 대시보드가 보임
- [ ] 우측 상단에 이메일이 표시됨
- [ ] "Welcome to Cloudflare" 메시지 확인

## 🆘 문제 해결

### 이메일이 안 와요
1. 스팸/정크 메일함 확인
2. 프로모션 탭 확인 (Gmail)
3. 2-3분 기다려보기
4. 이메일 주소 재확인

### 비밀번호를 잊어버렸어요
1. 로그인 페이지에서 "Forgot password?" 클릭
2. 비밀번호 재설정 이메일 받기

### 계정이 이미 있대요
→ 로그인하세요: https://dash.cloudflare.com/login
